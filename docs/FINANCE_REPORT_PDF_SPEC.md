# BigConnect AI — Finance Report PDF Redesign Specification

## Objective
Redesign the exported Finance Report PDF to match BigConnect AI invoice/receipt branding. The report must look like an official finance document, not a browser printout.

## Core Requirement
- Use a **dedicated FinanceReportPDFTemplate** on the backend
- Do NOT print the Finance dashboard UI
- Preview and download use the **same backend-generated PDF**
- Only `Content-Disposition` changes: `inline` for preview, `attachment` for download

## API Endpoints

```
POST /api/v1/admin/finance/reports/export
GET  /api/v1/admin/finance/reports/{reportId}/file?mode=inline
GET  /api/v1/admin/finance/reports/{reportId}/file?mode=download
```

## PDF Layout Structure
1. Page Header (BigConnect AI branding + report metadata)
2. Report Identity Section (name, type, period, currency, generated by)
3. Report Summary Strip (executive summary + filter chips)
4. KPI Summary Grid (3x2 compact grid)
5. Financial Breakdown (revenue by plan, gateway, status)
6. Transactions Table (professionally styled)
7. Revenue Summary (right-aligned totals box)
8. Notes / Audit Metadata
9. Footer (company info + page numbers)

## Page Setup
- A4 portrait, white background
- Margins: 28-36px
- Table headers repeat on page break
- Footer repeats on every page
- Page numbers: "Page 1 of 3"

## Export Request Payload
```json
{
  "reportType": "payments",
  "format": "pdf",
  "datePreset": "last_3_months",
  "filters": {
    "clientId": null,
    "paymentStatus": null,
    "gateway": null,
    "paymentMethod": null,
    "currency": "GHS"
  }
}
```

## Permission Rules
- Allowed: Super Admin, Platform Admin, Finance Admin
- Restricted: Viewer, Support Admin, Technical Admin

## Acceptance Criteria
- Matches invoice/receipt branding
- Solid colors only, no gradients
- No browser print headers
- KPIs in compact 3-column grid
- Professionally styled transactions table
- Includes selected filters and metadata
- Branded footer with page numbers
- Downloaded PDF matches preview exactly
- Multi-page handling with repeated headers
- Empty state renders professionally
- Supports PDF and Excel
- Backend enforces permissions
